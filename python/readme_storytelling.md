# 2강 데이터 타입

## 데이터 타입

```
Q. 데이터 타입을 왜 알아야 할까?
A. 컴퓨터 메모리 공간을 효율적으로 사용하고 연산 규칙을 틀리지 않기 위해서.

[전제] 데이터 타입에 따라 컴퓨터의 메모리 공간 사용과 연산 규칙이 달라진다.
[결론] 메모리 공간 효율적 사용 ^ 연산 규칙 올바르게 활용 → 데이터 타입 이해
[대우] ~데이터 타입 이해 → ~(메모리 공간 효율적 사용 ^ 연산 규칙 올바르게 활용)
```

```
데이터 타입 = 용량 + 유형
```

## 기본 데이터 타입

```
사과를 담기 위해서는 비닐봉지와 페트병 중 누가 필요할까?
물을 담기 위해서는 누가 필요할까?

현실 세계에서는 물, 사과, 모래, 쌀 등 굉장히 다양한 형태가 존재한다.
컴퓨터에서는 3가지 타입만 알면 된다. 이를 기본 데이터 타입이라 한다.

사과와 물을 한꺼번에 담아도 될까? 담을 수는 있다. 그러나 꺼림칙하다.
문자와 정수를 한꺼번에 담아도 된다. 그러나 연산할 때 조심해야 한다.

A1. 컴퓨터의 메모리는 한정되어 있고 데이터 타입에 따라 차지하는 용량이 달라지기 때문이다.
A2. 데이터 타입에 따라 컴퓨터가 연산하는 방법이 달라지기 때문이다.

데이터 타입을 보면 '용량'과 '유형'을 떠올려라.
```
```
a = "Jay"
b = 1990

type(a)
type(b)

# a의 타입: str
# b의 타입: int
```

## 변수 (Variables)

```
Q. 수학에서 쓰는 변수 == 컴퓨터에서 쓰는 변수
A. False. 컴퓨터에서 쓰는 변수는 데이터를 임시로 저장하는 "공간"을 뜻한다. 공간이기 때문에 변수를 사용하려면 용량과 유형을 고려해야 한다. 즉, 데이터 타입을 고려해야 한다.
```

```
a = "Jay"
b = 1990

# a는 문자형 변수이고, b는 정수형 변수이다.
```
```
c = str(1990)
user_input = input("태어난 연도를 입력하세요:")

# c는 문자형 변수이다.
# user_input은 [문자형/숫자형] 변수이다.
```

```
user_input = int(input("태어난 연도를 입력하세요:"))
user_age = 2017 - user_input

# 데이터 타입을 변경하는 것을 형 변환(conversion)이라 한다.
```
